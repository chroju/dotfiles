# タスク

直前の自分の応答を振り返り、ユーザーに確認すべきだったのに勝手に進めた点・推測で答えた点を `AskUserQuestion` で聞き直す。

## 実行ステップ

1. AskUserQuestionがdeferred toolとして名前だけしか見えていない場合は、先に `ToolSearch` を `select:AskUserQuestion` で呼んでスキーマをロードする
2. 直前のやり取りを振り返り、以下に該当する箇所を特定する
   - 複数のやり方がある中で、ユーザーの好みを聞かずに決めた
   - 要件・スコープが曖昧なまま推測で進めた
   - 副作用や影響範囲が大きい操作を確認なく実行しようとした
3. 該当箇所を `AskUserQuestion` で聞き直す
   - 選択肢として提示できる質問は必ず選択肢化する（自由記述に逃げない）
   - 既に進めてしまった作業がある場合は、聞いた答えに合わせて巻き戻す前提で質問する
4. 該当箇所がない場合は、そう答えて終わる（無理に質問を作らない）
